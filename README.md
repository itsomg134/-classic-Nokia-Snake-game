## ğŸ¯ Game Rules

1. The snake starts with a length of 1 segment
2. The snake moves continuously in the current direction
3. Eating food increases the snake's length by 1 segment
4. The game ends if the snake hits a wall
5. The game ends if the snake collides with itself
6. You cannot reverse direction (e.g., if moving right, you cannot immediately move left)
